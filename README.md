# K72 Website

A modern and visually engaging website built with React, Vite, Tailwind CSS, and GSAP animations. The project features immersive video backgrounds, smooth transitions, and a responsive user interface inspired by contemporary creative agency websites.

## Features

- Responsive Design
- GSAP Animations
- Video Hero Section
- Interactive Navigation Menu
- Smooth User Experience
- Project Showcase Section
- Modern UI/UX
- Optimized Vite Build

## Tech Stack

- React.js
- Vite
- Tailwind CSS
- GSAP
- React Router DOM

# Installation

git clone <repository-url>
cd k72
npm install
npm run dev
Build
npm run build
Deployment

The application is deployed on Vercel and configured for production-ready builds.
